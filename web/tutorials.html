Introduction to Python
Programs are just a set of instructions to get your computer to do something for you...In addition you will develop useful problem solving skills as you program...however careful you are...at some point you are likely to end up with a "bug" in your code(and it just don't run)..& you will need to develop techniques to identify and solve the problem. To write good programs, you need to learn the key commands...how to plan the key steps...build your program up...test...debug...& run.

Python is a high level programming language that is relatively easy to learn and a good springboard from which you can learn other programming languages. It is also used by professional software writers & people in research labs around the world. The python commands you write in your programs are intepreted by the computer & you can run programs via the IDLE(Python GUI) or using the commandline version.
