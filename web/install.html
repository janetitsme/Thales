Install Python

How to install Python
You can download Python from the download page at http://python.org for Windows. After the download completes, you can double click the file to start the installer. The default settings are fine to proceed . You keep clicking Next until the installation finishes.

Be sure to download Python 3 version(such as 3.4, 3.5) rather than a version (such as 2.5,2.6, or 2.7).

Python lets you make games that accept the text and display it. Pygame library adds the ability to program games with graphics, sound and animation. To install the pygame library, download the appropriate package from http://pygame.org/download.shtml or click here to download the Pygame library for Windows.

Why you should learn
1. Easy to Learn :Python was designed with the newcomer in mind. Python reads like kindergarten maths and is easy. The use of white space and common expressions has eliminated the need for tedious variable declarations and unattractive curly brackets. Python also requires less code to complete basic tasks, making it an economical language to learn. Python code is often 3-5 times shorter than Java.
2: Raspberry Pi :is a card-sized, inexpensive microcomputer that is being used for a surprising range of exciting do-it-yourself stuff such as robots, remote-controlled cars, and video game consoles. With Python as its main programming language, the Raspberry Pi is being used even by school children to build radios, cameras, arcade machines, and pet feeders! With Raspberry Pi mania on the uptrend, there are countless DIY projects, tutorials, and books to choose from online. These will help you branch out from your “hello world” starter programs to something you can truly be proud of.
